# Rock Paper Scissors

This is a simple Rock Paper Scissors game that I built in Python. The game runs in the terminal and lets you play against the computer as many times as you want. The computer makes a random choice each round, and the game keeps track of wins, losses, and ties until you decide to quit.

I created this project to practice Python fundamentals like loops, conditionals, user input, and using the `random` module.

## Features

* Play against the computer
* Random computer moves
* Tracks player wins, computer wins, and ties
* Handles invalid inputs
* Play multiple rounds until you quit

## Requirements

* Python 3.x

## How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/rock-paper-scissors.git
```

Go to the project folder:

```bash
cd rock-paper-scissors
```

Run the game:

```bash
python main.py
```

> Replace `main.py` with your Python file name if it's different.

## How to Play

* Enter **Rock**, **Paper**, or **Scissor** when prompted.
* The computer will randomly choose its move.
* The winner is displayed after each round.
* Enter **q** anytime to quit the game.
* When you quit, the final results will show the total wins, losses, and ties.
